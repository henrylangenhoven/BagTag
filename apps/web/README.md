# BagTag Web

Angular frontend for BagTag.

This app is currently scaffolded but still close to the Angular starter state. It is intended to hold both the owner experience and the public tag flow described in the root project docs.

## Commands

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Notes

- Default local URL: `http://localhost:4200`
- Routing and feature modules have not been built out yet
- See the root [`README.md`](../../README.md) and [`docs/architecture.md`](../../docs/architecture.md) for project context
